# Property Finder Website

A property finder website built using HTML, CSS, PHP, and JSON.

## Features
- Property listings
- Login and logout system
- Search page
- Ranking page based on ratings
- Reviews stored using JSON

## Technologies
- HTML
- CSS
- PHP
- JSON

## How to Run
1. Install XAMPP
2. Place the project folder inside `htdocs`
3. Start Apache
4. Open the project in the browser using localhost
